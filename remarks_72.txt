Подзапрос в SQL.
Когда создается временная таблица, которая служит в качестве критерием для дальнейшей выборки.
:::::
> ALL - значение больше максимального значения из выборки
< ALL - значение меньше минимального значения из выборки
< ANY - значение меньше максимального значения из выборки
> ANY - значение больше минимального из выборки (в диапазоне)
